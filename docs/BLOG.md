## 🌟 核心功能

### 1. 即時訂單通知 🔔

新訂單到達時：
- 右上角鈴鐺顯示紅點
- 頁面頂部顯示通知橫幅
- 播放提示音
- 點擊即可查看詳情

**價值：** 即時響應，提升服務品質

### 2. 靈活的配料系統 🧩

- 為每個店鋪設定配料列表
- 設定預設價格
- 在產品中可覆蓋價格
- 設定每筆訂單最多配料數量

**價值：** 靈活定價，吸引顧客

### 3. 飲品選項（冷飲/熱飲）🥤

- 勾選「提供冷飲」或「提供熱飲」
- 設定加價（可為 0）
- 顧客下單時可選擇

**價值：** 滿足不同季節和顧客需求

### 4. QRCode 掃碼點餐 📱

- 啟用桌號掃碼點餐
- 批量創建桌號
- 自動生成 QRCode
- 一鍵打印

**價值：** 減少人力成本，提升點餐效率

### 5. 回饋金系統 💎

- 消費累積回饋金
- 下次購物可折抵
- 可跨店使用
- 查看交易記錄

**價值：** 提升顧客回購率

### 6. 組合支付 💳

支援：現金、LINE Pay、街口支付、信用卡等

**價值：** 靈活支付，滿足不同需求

## 🚀 快速開始

### 店鋪管理者

1. **登入系統** → 點擊「商店管理者登入」
2. **設定店鋪** → 填寫店鋪資訊，設定訂單ID
3. **新增產品** → 填寫產品資訊，設定飲品選項和配料
4. **處理訂單** → 收到通知後立即處理

### 顧客

1. **註冊帳號** → 填寫姓名、Email、密碼
2. **瀏覽下單** → 選擇店鋪和產品，加入購物車
3. **結帳** → 選擇支付方式，完成訂單
4. **追蹤訂單** → 查看訂單狀態
